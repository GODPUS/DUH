Variables

(in _variables.scss)
Breakpoints
key is the name, first value in list is the min-width of the screen to trigger the breakpoint. second value in the list is the width of the .container class at that screen size. the breakpoint names will be used as the suffix on responsive classes.
$breakpoints: ('sm': (0, 100%), 'md': (768, 100%), 'lg': (960, 960px));
Columns
these are the different column amounts a row can have, the numbers will adjust the width of direct child column widths example: class="row cols-5"
$rowCols: (5, 12, 24);
the default number of columns a row has
$defaultRowColNum: 12;
Gutters
gutter classes are meant to be used in conjunction with col- classes to specify their left and right padding. child .row classes will inherit these and correct their margin-left and margin-right accordingly. example: class="col-6-sm gutter-15-md" (this would give a 15 pixel padding to this column on size md and up, and go back to the default on sm)
$gutterSizes: (0, 5, 10, 15, 20, 25, 30);
set default size of column classes child .row classes will inherit these and correct their margin-left and margin-right accordingly.
$defaultGutterSize: 10;

Grid

According to the settings above:
col-6-sm (of the default 12) with no gutter col-3-md center-horiz-md-only (of the specified 5) col-18-lg (of the specified 24)
col-6-sm (of the default 12) with no gutter col-3-md center-horiz-md-only (of the specified 5) col-4-lg (of the specified 24)

Javascript via data attributes

a combination of data-show-href, data-hide-href, data-toggle-href, data-scroll-href, data-tab-group are used to make dropdowns, accordions, tabs, modals, and scroll-to's
Dropdowns
  • if two elements have the same href they will update the other when one changes like with this select group and button group
  • a 'show' and a 'hide' event are triggered for both the button and the href, which you can listen and respond to via js like in demo.js
  • an .active class is also added to both the button and the href which you can use via css for styling and cssanimations
